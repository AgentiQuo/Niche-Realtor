# Niche Realtor

Niche Realtor is a multi‑agent, AI‑native real‑estate intelligence system.

## Architecture
- **Frontend**: Next.js + React + Tailwind
- **Backend**: Python (FastAPI)
- **Data**: Vector database for embeddings
- **Agents**: Ingestion, Analysis, Scout, Client Modeling, Matching Engine

## Core Components
- **Niche Ingestion**
- **Property Analysis**
- **Neighborhood Scout**
- **Client Modeling**
- **Matching Engine**

## Developer Guide
- **Frontend**: Located in `/frontend`
- **Backend**: Located in `/backend` (Runs FastAPI)
- **Data Models**: Defined in `/data-models`
- **Agents logic**: Built in `/agents`
- **Embeddings math**: Stored in `/embeddings`
- **API contracts**: Described in `/api/contracts.md`
